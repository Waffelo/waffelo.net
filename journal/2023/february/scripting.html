Scripting

26th February 2023

I wanted to learn Bash scripting, so I started scripting some tools that I would need. I initialized a new Git repository for my system scripts Woofy-Scripts. First I made a backup script for my system which copies all my files onto a selected directory, it can be a external disk. It also backs up my gpg private key.

I also made a post installation script for my system which installs yay and needed packages. It copies all files backed up by my backup script into my home directory. It recompiles Suckless software utilities like DWM or ST and installs Vim-plug.
